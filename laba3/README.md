## **Вариант 1: Зарплата охранника**

Доработайте программу расчёта зарплаты, реализовав:

### 1. **Работа с массивами:**

- Создайте массив для хранения данных по зарплатам за последние 6 месяцев
- Реализуйте возможность просмотра истории расчётов
- Добавьте расчёт средней зарплаты за период

### 2. **Использование функций:**

- Вынесите основные операции в отдельные функции:
  - `CalculateSalary` - основной расчёт
  - `ApplyBonuses` - применение премий
  - `ApplyPenalties` - применение штрафов
  - `CalculateTax` - расчёт налога
- Создайте функцию для вывода результатов в удобном формате

### 3. **Новые возможности:**

- Добавьте анализ динамики зарплат:
  - Вывод месяцев с максимальной/минимальной зарплатой
  - Расчёт процента изменения относительно предыдущего месяца
- Реализуйте поиск по истории (по сумме, по дате)

**Пример работы программы:**

```
=== Калькулятор зарплаты охранника ===

1. Рассчитать текущую зарплату
2. Просмотреть историю расчётов
3. Анализ динамики зарплат
4. Поиск в истории
5. Выйти

Выберите действие: 3

=== Анализ динамики ===

Средняя зарплата за 6 месяцев: 28 450 руб.
Максимальная зарплата: 32 100 руб. (Март 2023)
Минимальная зарплата: 25 300 руб. (Январь 2023)
Изменение за последний месяц: +5.2%

1. Вернуться в меню
2. Выйти
```

**Требования к реализации:**

- Использовать одномерные массивы для хранения истории
- Все основные операции должны быть вынесены в отдельные функции
- Реализовать защиту от переполнения массива
- Добавить проверку корректности данных при сохранении в историю
